This is the first post on my new site. I set it up mostly as a way to:

- keep track of things I learn while building software,
- have a single place to point people when they ask what I've been working on,
- and force myself to write more. Writing is the best debugger for thinking.

You can write Markdown here — headings, lists, `inline code`, bold/italics, and fenced code blocks all work:

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

Posts are MDX files, so you can also drop React components in if you need them.
